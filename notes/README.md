# notes

トピック別の個人ノート。1トピック1フォルダで運用する。

## 構成

```
notes/
└── {トピック名}/
    ├── summary.md           # トピック全体の要約・年表
    ├── articles/            # 個別記事・論文の要約
    │   └── {タイトル}.md
    └── refs.md              # 参考リンク一覧
```

## 記事メモのフォーマット

各 `articles/{タイトル}.md` には最低限以下を書く：

- **出典**: URL / DOI / arXiv ID（一次情報を必ず）
- **著者・発行年**
- **要点**: 3〜5行
- **詳細**: 手法・結果・考察
- **所感**: なぜ面白いと思ったか

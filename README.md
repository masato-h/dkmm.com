# dkmm.com

情報収集していて面白かったやつとか、学会関連のメモを蓄積する個人ノートリポジトリ。

## 構成

- `notes/` — トピック別ノート（読んだ論文・記事・調査結果）
- `conferences/` — 学会・カンファレンス関連メモ

## 関連リポジトリ

- 研究テンプレート: [masato-h/research_template](https://github.com/masato-h/research_template)
- DSPy + LLM as a Judge 実験: [masato-h/dspy-experiments](https://github.com/masato-h/dspy-experiments)
